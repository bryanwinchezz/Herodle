# 🦸 Herodle

Um jogo de adivinhar personagens de super-heróis — pense no Wordle, mas para o universo geek! Teste seus conhecimentos sobre heróis e vilões da Marvel, DC e muito mais.

## 📋 Sobre o Projeto

O **Herodle** é um jogo de adivinhação inspirado no Wordle, mas com foco no universo dos super-heróis. A cada tentativa, o jogador recebe dicas sobre características do personagem misterioso — origem, poderes, universo, afiliação — e tenta acertar quem é antes de esgotar as tentativas.

## 🎮 Como Jogar

1. Abra o jogo no navegador
2. Digite o nome de um personagem de super-herói
3. Receba dicas sobre as características do personagem misterioso
4. Use as dicas para refinar suas tentativas
5. Acerte o personagem antes de acabar as tentativas!

**Dicas comuns incluem:** universo (Marvel/DC), gênero, poderes, afiliação (Avengers, X-Men, Liga da Justiça...), origem do personagem e muito mais.
